# AI Knowledge Library - Deutsch (de_DE)

Dies ist die ins Deutsche übersetzte Dokumentation für die AI Knowledge Library.

## Verwendung des Installationsprogramms in dieser Sprache
Um das Installationsprogramm auf Deutsch auszuführen, setzen Sie die Umgebungsvariable `LANG` oder wählen Sie diese während der Einrichtung aus:

```bash
export LANG=de_DE.UTF-8
./ai_installer install
```

## Inhalt übersetzen
Der KI-Agent ist so konfiguriert, dass er englische Fähigkeitsdefinitionen und Eingabeaufforderungen ins Deutsche übersetzt, wenn dieses Modul aktiv ist.
