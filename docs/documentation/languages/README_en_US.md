# AI Knowledge Library - US English (en_US)

This is the US English localized documentation for the AI Knowledge Library.

## Using the Installer in this Language
To run the installer in US English, set the `LANG` environment variable or select it during setup:

```bash
export LANG=en_US.UTF-8
./ai_installer install
```

## Translating Content
The AI Agent is configured to translate English skill definitions and prompts into US English when this module is active.
