# Cyber Security Directives

## Security Posture
* Implement zero trust architecture across all internal and external communication channels.
* Automatically scan all repositories for exposed credentials and PII before merging code.
* Regularly rotate access keys and encrypt sensitive data at rest and in transit.
* Maintain comprehensive audit logs for all administrative actions.

## Privacy and PII Guidelines
* Never record phone numbers, physical addresses, or any highly sensitive PII in repository files.
* Only names and email addresses are permitted for identification purposes.
