# Left 4 Dead 2 Scripting

Write, debug, and optimize scripts for Left 4 Dead 2 using SourcePawn (for SourceMod plugins) and VScript (Squirrel language) for map-specific or mutation logic.

## Key Focus Areas
- **SourcePawn**: Writing efficient plugins, hooking events (e.g., `player_death`, `weapon_fire`), manipulating entities, and managing memory/handles.
- **VScript**: Using the Squirrel language to create custom mutations, director scripts, and map logic.
- Entity manipulation and navigation meshes (NavMesh).
- Debugging memory leaks and logic errors in plugins.
