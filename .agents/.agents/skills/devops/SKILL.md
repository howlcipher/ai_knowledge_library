# DevOps Practices

## Pipeline and Infrastructure
* Treat infrastructure as code utilizing tools like Terraform or Ansible.
* Design deployment pipelines to support zero downtime rolling updates.
* Ensure container images are minimal and scanned for vulnerabilities prior to deployment.
* Standardize build processes across all environments from development to production.
