# AI Knowledge Library - Arabic (ar_SA)

This is the Arabic localized documentation for the AI Knowledge Library.

## Using the Installer in this Language
To run the installer in Arabic, set the `LANG` environment variable or select it during setup:

```bash
export LANG=ar_SA.UTF-8
./ai_installer install
```

## Translating Content
The AI Agent is configured to translate English skill definitions and prompts into Arabic when this module is active.
