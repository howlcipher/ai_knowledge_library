# AI Knowledge Library - العربية (ar_SA)

هذه هي الوثائق المترجمة إلى اللغة العربية لمكتبة المعرفة بالذكاء الاصطناعي (AI Knowledge Library).

## استخدام برنامج التثبيت بهذه اللغة
لتشغيل برنامج التثبيت باللغة العربية، قم بتعيين متغير البيئة `LANG` أو حدده أثناء الإعداد:

```bash
export LANG=ar_SA.UTF-8
./ai_installer install
```

## ترجمة المحتوى
تم تكوين وكيل الذكاء الاصطناعي لترجمة تعريفات المهارات والمطالبات الإنجليزية إلى اللغة العربية عندما تكون هذه الوحدة نشطة.
