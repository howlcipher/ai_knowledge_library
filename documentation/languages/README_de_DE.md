# AI Knowledge Library - German (de_DE)

This is the German localized documentation for the AI Knowledge Library.

## Using the Installer in this Language
To run the installer in German, set the `LANG` environment variable or select it during setup:

```bash
export LANG=de_DE.UTF-8
./ai_installer install
```

## Translating Content
The AI Agent is configured to translate English skill definitions and prompts into German when this module is active.
