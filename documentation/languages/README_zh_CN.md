# AI Knowledge Library - Chinese (Simplified) (zh_CN)

This is the Chinese (Simplified) localized documentation for the AI Knowledge Library.

## Using the Installer in this Language
To run the installer in Chinese (Simplified), set the `LANG` environment variable or select it during setup:

```bash
export LANG=zh_CN.UTF-8
./ai_installer install
```

## Translating Content
The AI Agent is configured to translate English skill definitions and prompts into Chinese (Simplified) when this module is active.
