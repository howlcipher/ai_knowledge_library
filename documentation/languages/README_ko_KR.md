# AI Knowledge Library - Korean (ko_KR)

This is the Korean localized documentation for the AI Knowledge Library.

## Using the Installer in this Language
To run the installer in Korean, set the `LANG` environment variable or select it during setup:

```bash
export LANG=ko_KR.UTF-8
./ai_installer install
```

## Translating Content
The AI Agent is configured to translate English skill definitions and prompts into Korean when this module is active.
