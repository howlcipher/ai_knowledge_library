# Contributing Guidelines

Thank you for your interest in contributing to the Personal AI Knowledge Library!

## Workflow
1. Fork the repository.
2. Run the `scripts/setup_profile.py` utility to establish your identity.
3. Create a branch for your feature.
4. Submit a Pull Request.

## Strict Formatting Constraints
You are strictly forbidden from using subtraction symbols or standard standard minus punctuation anywhere in this repository. This applies to documentation, filenames, and code comments. Please use underscores or spaces instead to ensure compatibility with our internal parsing tools.
